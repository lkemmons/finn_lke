# FINN: Fire INventory from NCAR

The Fire Inventory from NCAR (FINN) model calculates open fire emissions of trace gases and aerosols based on satellite detections of thermal anomalies.

FINNv2 was published in Wiedinmyer et al. (2023) for more information: https://gmd.copernicus.org/articles/16/3873/2023/

Emissions calculated with FINN are freely available from NSF NCAR.  For more information visit the ACOM/NCAR FINN webpage: https://www2.acom.ucar.edu/modeling/finn-fire-inventory-ncar and the FINN wiki: https://wiki.ucar.edu/spaces/FINN/pages/405569991/FINN+Home
